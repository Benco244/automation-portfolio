01

Automated Job Aggregation & Publishing Pipeline

End-to-end workflow orchestration (n8n)

The Problem

Manual job sourcing and publishing across multiple employer websites — slow, inconsistent, unscalable.

The Solution

Built a complete n8n automation pipeline that:

  • Ingests pre-scraped job data from multiple sources
  • Filters for student-friendly and part-time roles
  • Deduplicates records using external links
  • Maps data to platform API schema
  • Posts jobs automatically via authenticated REST API
  • Captures success/failure responses
  • Generates execution reports and email notifications

Impact

Eliminated manual job posting workflow entirely. Created reusable orchestration framework for future data pipelines. Enabled scheduled and on-demand publishing at scale.

Tools: n8n · JavaScript · REST APIs · Data transformation
02

Multi-Employer Job Scraping System

Data extraction and API integration (Node.js)

The Problem

Each employer website had different structures, pagination logic and job detail layouts — making unified extraction difficult.

The Solution

Built modular Node.js scraping scripts that:

  • Handle site-specific DOM structures and pagination
  • Extract salary, job type, location, deadlines and external links
  • Normalize raw data into a unified JSON schema
  • Submit transformed jobs to platform API with response validation
  • Log success/failure for each posting

Impact

Automated extraction from multiple employer portals. Standardized job data across sources. Achieved reliable posting with verified HTTP 200 responses.

Tools: Node.js · Puppeteer · REST APIs · VS Code

AI-Powered CV Structuring Tool

AI-assisted document automation

The Problem

Job seekers struggle to convert unstructured personal information into professionally formatted CVs suitable for ATS systems and recruiters.

The Solution

Built a web-based tool that:
